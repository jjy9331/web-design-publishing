

뷰CLI를 설치하려면 노드제이스가 설치되어있어야한다.
(CLI: Command Line Interface - 커맨드창에서 명령어로 특정 동작을 수행할 수있는 도구)
nodejs.org 에서 다운로드 왼쪽 버튼 클릭하여 설치

- 명령 프롬프트(키보드의 윈도우키+R, cmd입력후 엔터)
   또는 [시작] – [모든 프로그램] – [보조 프로그램] – [명령 프롬프트] 

- 명령 프롬프트(터미널) 에서 node 입력후 엔터 쳤을때 버전설명 나오면 정상설치된것





0. 프로젝트 디렉토리 경로로 들어가기

명령 프롬프트에서

cd.. (상위경로 이동)
d: (d드라이브 이동)
cd 폴더명 (하위폴더 이동)

   (명령 잘 안될때 [명령 프롬프트] 바로가기에 오른쪽 클릭후, [관리자 권한으로 실행 (A)] 선택 )





1. 설치
명령 프롬프트에서
프로젝트 폴더로 들어간다

npm install vue-cli -global


2. 실행 확인
명령 프롬프트
vue


3. 뷰CLI 프로젝트 생성
명령 프롬프트
vue init webpack-simple

질문 모두 y
마지막 sass만 n

명령 프롬프트
npm install


4. 실행
명령 프롬프트
npm run dev
