웹표준이란

웹표준이란 웹에서 표준으로 사용되는 기술을 말하며, 국제 표준화 단체인 W3C( World Wide Web Consortium )가 권고한 표준안에 따라 목적과 방법에 맞게 사용하는 것이라 할수 있습니다. W3C 표준안에 맞지 않는 비표준 요소를 사용하거나 특정 브라우저에만 최적화 되도록 제작하면, 다른 브라우저에서 원할한 접근이 어려워질 수 있습니다. 따라서 웹표준이란 우리가 만든 사이트에 접속한 사용자가 어떠한 운영체제나 브라우저를 이용하더라도 같은 결과를 볼 수 있도록 웹페이지를 만든다는 의미도 담겨있습니다.

웹표준의 장점

마크업 용량감소
웹표준을 준수하여 CSS와 HTML 문서를 분리하여 제작하게 되면 불필요한 코드량이 줄어들기 때문에 페이지 로딩 시간이 빨라집니다.
컨텐츠와 CSS분리
웹표준을 이용해 HTML로 구조화된 컨텐츠와 CSS를 따로 분리해 놓으면 스타일시트 내용 수정만으로 손쉽게 모든 디자인의 수정이 가능해 집니다.
검색 엔진에 대한 최적화 방안
웹표준을 이용하여 HTML 소스를 중요도에 맞게 구조화하고 ,깨끗하게 정리된 소스를 만들어내면 검색로봇이 올바르게 검색할수 있습니다.
웹 접근성 용이
웹 접근성이란 모든 사용자들이 웹사이트의 컨텐츠를 쉽게 인식하고 서비스를 편리하게 이용할 수 있도록 만드는 것을 말하며, 웹표준을 이용해 작성한 문서는 대부분의 브라우저, 휴대폰, 장애인 지원용 소프트웨어 등에서 쉽게 읽을 수 있습니다.
